PHASE 1 — MVP LANDING Landing page để seed & validate
Tuần 3–5 · Sprint 2
Mục tiêu
Đo quan tâm thực sự trước khi code full
Output
Landing page đẹp + 1 bài test thử beta
KPI validate
500+ lượt làm test · CTR social >3%
Dev tasks
  • Landing page (Webflow / Next.js đơn giản)
  • 1 bài test beta hoàn chỉnh (quiz + kết quả đẹp)
  • Kết quả có nút share ảnh PNG (ưu tiên #1)
  • Logic tính điểm → map ra type kết quả
  • Thu email / Zalo OA để remarketing
  • GA4 + Hotjar free (heatmap & recording)
  • Deploy Vercel + custom domain
Content tasks
  • 15–20 câu hỏi bài test beta (dùng Claude AI)
  • 4–8 types kết quả cực relatable
  • Copy landing page: tagline, CTA, mô tả
  • Design result card đẹp để share (Artist)
  • Brief visual brand: màu, font, tone
Phân công đội
PO: brief content, QA, điều phối BA/DEV: backend quiz engine FE Dev 1: landing page FE Dev 2: quiz flow + result Artist: result card, brand visual 4 Content: câu hỏi, mô tả kết quả
Ưu tiên tuyệt đối: Share card đẹp → nội dung relatable → giao diện → quiz logic. Share card chính là viral engine, làm trước.
PHASE 1 — SEEDING Social validation & Go/No-Go
Tuần 4–6 · Sprint 3
500+ lượt làm test Share rate >20% CTR social >3% Sentiment positive >70% 200+ email/Zalo
Seeding tasks
  • 5–10 video TikTok/Reels teaser (15–30s)
  • Seeding Threads: post kết quả tự nhiên
  • Seeding 10–15 FB Groups self-dev VN
  • 2 video YouTube faceless (5–15 phút)
  • Schedule đăng: TikTok 3–5 vid/tuần
Tool AI cho video
  • CapCut AI: dựng video TikTok nhanh
  • HeyGen: avatar đọc kết quả test
  • ElevenLabs: giọng đọc tiếng Việt
  • Claude: viết script video, comment seeding
  • Buffer/Hootsuite: lên lịch đăng tự động
GO / NO-GO decision: Sau 2–3 tuần seeding, họp toàn đội review số liệu. Đạt KPI → Phase 2 build proper. Không đạt → pivot concept hoặc đổi bài test trước khi đầu tư tiếp.
PHASE 2 — BUILD PROPER Xây platform thật sự sau khi validate
Tháng 2–3 · Sprint 4
Tech stack
Next.js + Supabase + Vercel
KPI target
5.000 users · 30% quay lại
Điều kiện bắt đầu
Phase 1 đạt GO decision
Must-have features
  • Thư viện 5–10 bài test đa dạng chủ đề
  • Đăng nhập Google / Facebook (Supabase Auth)
  • Profile: lưu lịch sử, kết quả các bài test
  • Share card tự động xuất ảnh PNG
  • So sánh kết quả với bạn bè (link invite)
  • SEO cho từng trang bài test
Nice-to-have features
  • Dark mode
  • Combo test (MBTI + Love Language)
  • Push notification khi có bài test mới
  • Bảng xếp hạng type trong cộng đồng
  • Chatbot gợi ý bài test phù hợp
